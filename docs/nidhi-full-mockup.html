Design Spec · Architecture
Theme: Agni (अग्नि) — warm gold on deep ocean blue-black. Distinct from catppuccin/tokyo night/dracula. Color downsampling handled automatically by colorprofile (built into BubbleTea v2). All colors in CSS vars → Go constants.

Layout contract:
Every screen = hbar (status) + content (varies) + fbar (keybinds + mode badge).
Content is one of: full-width list, vertical split (list+preview), horizontal split (tree+diff), overlay (modal/help).
LipGloss: JoinVertical for list/preview. JoinHorizontal for tree/diff. Canvas.Layer for modals.

Keyboard hierarchy (3 tiers):
Tier 1 (single key): j k g G a p d n r e / ? — covers 95% of use
Tier 2 (double key): fb fs (filter), Shift+J/K (reorder) — power features
Tier 3 (ctrl+key): ^d ^u ^p ^a ^n — scroll, patch, bulk select

Navigation flow:
LIST ⇄ PREVIEW (⇥ toggle) → DETAIL (⏎, esc back) → actions branch to CONFIRM / RENAME / SEARCH / EXPORT
Every overlay/mode returns to previous screen with esc. No dead ends.

Top 5 features (differentiators):
1. Conflict preview — merge-tree dry-run before apply/pop. Nobody else does this.
2. Deep search — fuzzy across messages + filenames + diff content. "Where's my code?"
3. Export/Import sync — Git 2.51 stash export with one-key push to remote.
4. Rename — inline message edit via drop+store. Fixes the #1 stash UX problem.
5. Undo — z to recover dropped stashes via reflog. Safety net for destructive ops.

OOTB defaults:
Auto-detects repo, branch, git version. Export ref defaults to refs/stashes/$USER. Stale detection at 14 days. --keep-index on by default for new stashes. Auto-generates readable messages from diff stats when user doesn't provide one. Config via ~/.config/nidhi/config.toml or git config --get nidhi.*.
